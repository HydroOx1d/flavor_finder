# Flavor Finder Frontend


### Get started

Clone the project

```bash
git clone https://github.com/HydroOx1d/react-boilerplate.git
```

Install the dependencies

```bash
npm install
#or
yarn install
```
Start the project

```bash
npm run dev
#or
yarn dev
```

Build the project

```bash
npm run build
#of
yarn build
```

Start tests

```bash
npm run test
#or
yarn test
```

Start linter

```bash
npm run lint
#or
yarn lint
```
